智慧手機連網控制家電應用

指導老師：馬立山 老師
建國科技大學 電子工程系

簡介

  本作品設計一個智慧家庭監控系統，主要以Arduino為家電控制核心，整合手機App、物聯網雲端平台及家電設備，透過網際網路連線到雲端，使行動裝置與家電設備能相互通訊， 配合攝影機傳回的即時影像，讓使用者可利用App進行遠端控制家電設備，也能即時監控家庭的狀態。此外，本作品利用手機GPS定位資料，結合大圓距離公式的計算，判斷家人是否超出設定的距離，當家人超出距離時，系統會即時通知使用者手機，以達到警示的作用， 另外在當日早上會立即收到天氣預報的訊息。

功能

1.    遠端控制：透過手機App可以直接遠程控制居家的家電設備。
2.    遠端監視：透過手機App可以直接遠程監視居家的情況。
3.    電子柵欄：透過手機App設定家人行走距離，當家人超出距離時，會即時發送LINE訊息至使用者手機，並且可查詢家人的所在位置。
4.    社群推播：固定時間發送天氣預報至使用者手機。

智慧電子朝拜系統一媽祖e起來

106年全國微電腦應用系統設計創作競賽
信號處理與通訊類 佳作

簡介

  本作品設計一個智慧電子朝拜系統，共有四個場景，主要應用於媽祖繞境、居家、廟宇拜拜等環境，輔助無法遶境的信徒能有遶境時的氛圍，而在繞境信徒也能有全新的體驗，同時加深神明與信徒之間關係。 在媽祖出巡時本系統可以用來尋找神轎位置、求籤、情緒分析與擲筊等，並且會依據當地溫度提醒繞境的信徒補充水分， 在家裡的信徒可利用VR眼鏡、手機、腳踏車等物品來增加遠端遶境的臨場感， 而在廟裡拜拜的信徒還可以利用手機來送出留言，並觀看廟口外的電視牆體驗遶境時的氛圍。

功能

本作品有四個場景，主要應用於媽祖繞境、居家、廟宇拜拜等環境，其功能如下：

1.    出巡朝拜系統

(1).    透過神轎上的溫濕度感測器與GPS定位，會依據神轎位置提醒信徒將抵達位置，同時依據當地的溫度通知信徒要補充水分。此外，也可藉由App進行社工留言，若信徒留言情緒較為負面，會給予適當的幫助，反之，會給予祝福的話。
(2).    當神轎抵達定點後，可使用手機掃瞄神轎，並在手機顯示出3D媽祖影像，在3D媽祖神像前可進行擲筊、抽籤、解籤等動作。

2.    千里眼/順風耳系統

(1).    廟口電視上播放媽祖遶境的直播影片。
(2).    廟口電視上播放媽祖FB粉絲頁的留言。

3.    遠端遶境系統

本作品於手機上提供媽祖繞境影片，可藉由腳踏車轉速控制播放速度，同時可配戴VR眼鏡來增加遠端遶境的臨場感。

4.    遠端朝拜系統

本作品於手機上提供媽祖繞境影片，可配戴VR眼鏡來體驗遠端朝拜，亦可利用社工關懷之App問籤。

4x4鍵盤控制RGB燈

介紹

使用鍵盤掃描方式讀取4x4鍵盤值來控制RGB燈的亮度及顏色，並加上LCM顯示目前的顏色及亮度。

  • 當按鍵0~7被按下時更改RGB燈顏色
  • 當按鍵A被按下時可輸入更改數值
  • 當按鍵B被按下時確定更改RGB燈亮度
  • 當按鍵C被按下時減少10%亮度
  • 當按鍵D被按下時增加10%亮度
  • 當按鍵F被按下時關閉RGB燈

    • 電路圖

    程式碼

    上傳溫濕度到雲端

    介紹

    使用8051的Uart功能對Esp8266(具有WiFI MCU)下At command命令讓8051能連上雲端(ThingSpeak)， 並且將讀取到的溫濕度數據以每隔20秒時間上傳至雲端。

    電路圖

    程式碼

    PWM調光

    介紹

    利用51單晶片上的Timer產生PWM控制LED燈亮度，並且使用2顆按鈕控制Duty cycle， 使LED燈的亮度能增加或減少，再搭配7段顯示器顯示目前LED的亮度。

  • 當按鍵1被按下時減少10%亮度
  • 當按鍵2被按下時增加10%亮度

    • 電路圖

    程式碼

    溫度警報器

    介紹

    使用4x4鍵盤設定溫度上限，並加上LCM顯示設定上限與目前溫度，當溫度超過上限時則會觸發蜂鳴器。

  • 當按鍵A被按下時可輸入更改數值
  • 當按鍵B被按下時確定更改溫度上限
